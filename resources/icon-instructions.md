# Icon Creation Instructions

## For App Icon

### Windows (icon.ico)

- Create a 256x256px PNG image with:
  - Health/wellness theme (heart, timer, activity symbols)
  - Clear and recognizable design
  - Bright, friendly colors
- Convert to .ico format using online tools like:
  - https://convertio.co/png-ico/
  - https://icoconvert.com/
- Include multiple sizes in .ico: 256x256, 128x128, 64x64, 48x48, 32x32, 16x16
- Save as: `resources/icon.ico`

## For Tray Icon

### Windows Tray (icon-tray.png)

- Create a 16x16px or 32x32px PNG
- Simple, minimalist design (works well at small size)
- Monochrome or simple colors
- High contrast for visibility
- Save as: `resources/icon-tray.png`

## Quick Design Ideas

### Option 1: Timer + Heart

- Circular clock face with heart in center
- Colors: Blue gradient (#3b82f6 to #8b5cf6)

### Option 2: Activity Symbol

- Stylized person stretching/exercising
- Colors: Green to blue gradient

### Option 3: Health Cross

- Modern health cross with rounded corners
- Timer tick marks around it
- Colors: Teal/cyan theme
